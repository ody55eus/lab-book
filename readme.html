Readme

Laboratory Book

This is my Lab Book for the magnetic fluctuation measurements. Here, I combine the documentation of all gathered data and used software to acquire and analyze the data. This software includes EVE, spectrumanalyzer, Ana, and Jupyter Notebooks. The links above point to my online GitLab server, which is available for reviewers with provided credentials.

The code can be tested in a local Jupyterlab.
