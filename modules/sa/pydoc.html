class SpectrumAnalyzer(**kwarg)[source]

Bases: object

Takes a timeseries and returns the Fourier Transformation.

This class calculates the fourier transform of a timesignal and computes the power spectral density of the signal. It also calculates the second spectrum and a histogram.

Caution

Don’t use Shifting Method together with the second spectrum!!

Parameters

  • timesignal (numpy.ndarray, optional) – the raw timesignal. If not given, filepath is used to retrieve information.

  • samplingrate (int, optional) – the rate in Hz for the timesignal. If not given, filepath is used to retrieve information.

  • averages (int, optional) – the number of chunks in the timesignal. If not given, filepath is used to retrieve information.

  • filepath (str, optional, default: None) – If given, it readsout the timesignal, rate and avg from *.hdf5 file Only used if timesignal, samplingrate and averages are not explicit given.

  • first_spectra_highest_frequency (int, optional, default: 128) – the highest frequency for the first spectrum

  • chunksize (int, optional, default: 1024) – the size of a single chunk

  • factor (numpy.ndarray, optional, default: None) – a factor to multiply the spectrum with (e.g. -> \(S_R/R^2\))

  • short_mode (bool, optional, default: False) – are we calculating a short or long first spectrum?

  • shifting_method (bool, optional, default: False) – Enables the Shifting Method.

  • filter_type (str, optional, default: lowpass.) – The type of filter: [‘lowpass’, ‘highpass’, ‘bandpass’, ‘bandstop’]

  • filter_order (int, optional, default: 6) – The order of the filter.

  • filter_analog (bool, optional, default: False (digital filter)) – Usage of analog or digital filter.

  • passband_factor (int, optional, default: 1) –

  • prefixes (dict, optional, default: None) – Prefixes for saving.

  • downsample (bool, optional, default: True) – If we should downsample the signal for the given frequency.

  • save_spectra (dict, optional, default: None) – Boolean values for different kind of saving operations

calculate_first_spectrum(timesignal, werte_nr=400, plot_every=True)[source]

Calculate First Spectrum and the corresponding Frequency Axis.

Important

  • Filters the time signal

  • Downsamples filtered_timesignal (if downsample == True).

  • Subtracts mean value (offset).

  • Applies window function (hanning).

  • Calculates fft.

  • Normalizes First Spectrum.

See additional info for normaization of first spectrum:

  • https://holometer.fnal.gov/GH_FFT.pdf

  • https://holometer.fnal.gov/GH_FFT.pdf#equation.24

  • https://holometer.fnal.gov/GH_FFT.pdf#equation.5

  • http://stackoverflow.com/questions/15147287/numpy-wrong-amplitude-of-fftd-array

  • http://www.mikrocontroller.net/attachment/16949/windows.pdf - some more information about window functions

Compare FFT with measured spectrum
first_spectrum_scipy = (abs(fft(final_array, axis=1))/np.sqrt(length_for_hanning))**2

       np.allclose(0, test, atol=1e-10),
       np.allclose(0, test, atol=1e-10))

cut_timesignal(number_of_first_spectra=1024, werte_nr=400, save_avg=False)[source]

Cuts the timesignal into pieces (before FFT). Cuts the timesignal in number_of_first_spectra splits and calculates the first spectrum out of it.

Important

This function effectively calculates the averaged spectrum. It calculates the number of timerelated spectra and averages them.

Caution

This is a generator and not a function! This means that it behaves like a iterator so you can loop over it.

Example
1for spectrum, frequency_span_array, k in cut_timesignal():
2    if k == number_of_first_spectra:
3        plot(frequency_span_array, spectrum, name='Averaged')
4    else:
5        plot(frequency_span_array, spectrum, name=k)
